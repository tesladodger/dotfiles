# Neovim configuration

## Prerequisites

- neovim
- fzf
- ripgrep
- xsel or xclip

## Installation 

Run `install` to create a symbolic link to this repo.

To install the plugins, open `lua/tesla/packer.lua`, `:source` it and
`:PackerSync`. 

You might have to sync twice.

Sync three times just to be sure.
